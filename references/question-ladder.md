# Question Ladder

Use this file when the user wants the full multi-layer research flow rather than a short answer.

## Layer 1: Supertrend

- What is the actual buildout?
- Who is spending?
- What hardware / infra must physically scale with that spend?
- Which parts are already fully consensus?
- What is the concrete machine or deployed system here, not the abstract theme?
- If you had to point at one rack, pod, module, or plant, what is it?

## Layer 2: Stack Mapping

- What are the 6-9 layers from materials to deployed system?
- Which layer gets paid first?
- Which layer gets paid last?
- Which layer has the least slack?
- Which component is most impossible to substitute quickly?

## Layer 3: Bottleneck Detection

- If demand doubles, what breaks first?
- Is this a capacity problem, qualification problem, yield problem, or materials problem?
- How many credible substitutes exist?
- Can customers dual-source it?
- Is there evidence that capacity is already allocated?
- If the main supplier stopped tomorrow, would the delay be weeks, quarters, or years?

## Layer 4: Mispricing Test

- Is the market focused on the obvious leader instead of the narrow supplier?
- Is this a first-order bottleneck or a second-/third-order bottleneck?
- Does the market cap reflect its role in the chain?
- Is the thesis early but real, or only imaginative?

## Layer 5: Evidence Test

- What did management actually say?
- Which customer / supplier tie is confirmed, and which one is inferred?
- Are multiple companies describing the same pain point?
- Is there a timestamped external proof point from reports or industry news?
- Is this proof strong, medium, or weak based on source quality?

## Layer 6: Positioning Test

- Is this a proven executor, a bottleneck supplier, or an early disruptor?
- Should this be a heavy weight, a basket weight, or a tiny optional position?
- What signal would justify adding?
- What signal would justify walking away?

## Layer 7: Follow-Up Drilldown

Only after the direction is built:

- Which lower-market-cap companies sit closest to the bottleneck?
- Which names are getting paid now versus later?
- Which company has the cleanest setup?
- Which company has the biggest upside but the weakest proof?
